Ejercicios Intermedios

1. Cambio de Familia de Fuentes: Selecciona un elemento en tu página web y cambia su familia de fuentes utilizando la propiedad font-family.

2. Ajuste del Tamaño del Texto: Modifica el tamaño del texto en varios elementos de tu página utilizando la propiedad font-size.

3. Negrita y Cursiva: Aplica estilos de negrita y cursiva a diferentes partes de tu texto utilizando la propiedad font-weight y font-style.

4. Espaciado entre Caracteres: Ajusta el espaciado entre caracteres en un elemento específico utilizando la propiedad letter-spacing.

5. Altura de Línea: Cambia la altura de línea en un párrafo o encabezado utilizando la propiedad line-height.

6. Alineación de Texto: Experimenta con la alineación del texto en diferentes elementos usando la propiedad text-align.

7. Uso de Fuentes Externas: Integra una fuente externa de Google Fonts o cualquier otra fuente web y aplica esta fuente a tu página.

8. Combinación de Fuentes: Crea una combinación de fuentes para diferentes tipos de elementos en tu página (encabezados, párrafos, enlaces, etc.).

9. Estilo de Lista de Enlaces: Diseña una lista de enlaces donde cada enlace tiene un estilo de fuente diferente (tamaño, color, estilo).

10. Creación de un Bloque de Cita con Estilo: Crea un bloque de cita (<blockquote>) y estilízalo con una fuente y un tamaño de texto distintos al resto del contenido.
